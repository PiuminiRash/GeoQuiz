# GeoQuiz Android App

GeoQuiz is a simple Android quiz application built using **Java** and **Android Studio**.  
The app allows users to answer **True or False questions**, track their progress, view their accuracy, and switch between multiple languages.

---

## 📱 Features

- True / False quiz questions
- Hint button to show the correct answer
- Quiz progress bar
- Accuracy percentage calculation
- Next and Previous question navigation
- Language support:
  - English
  - Sinhala
  - Tamil
- Call button integration
- State persistence during screen rotation

---

## 🛠 Technologies Used

- Java
- Android Studio
- Android SDK
- XML Layouts

## ⚙️ How It Works

1. The application loads questions from the question bank.
2. Users answer using **True** or **False** buttons.
3. The app calculates **accuracy percentage**.
4. The **progress bar updates** as questions change.
5. The **Hint button highlights the correct answer**.
6. Users can switch app language using the language buttons.
